# Medical Cost Prediction Web App

This project is an end-to-end Data Science application that predicts medical insurance costs based on user inputs.

## Features
- Data preprocessing using Pandas
- Machine learning model using Scikit-learn
- Model serialization with Pickle
- Web application using Flask
- Interactive user interface with HTML

## Input Parameters
- Age
- Sex
- BMI
- Number of Children
- Smoking Status
- Region

## Output
Predicted medical insurance cost.

## Technologies Used
Python
Pandas
Scikit-learn
Flask
HTML

## How to Run

1. Install dependencies
pip install flask numpy scikit-learn pandas

2. Run application
python app.py

3. Open browser
http://127.0.0.1:5000
